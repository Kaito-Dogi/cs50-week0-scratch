### コンピュータが直接理解できる言語
- コンピュータは2進数（binary）しか扱えない。
- オンオフでbitを表現するため。
- 現代のコンピュータはトランジスタ（無数のスイッチ）を持つ。

### 文字の表現
- 数字（2進数）と文字を対応させる。

### ASCII
- 8 bitで一文字を表現する。
- 8 bit（1 byte）で256種類（2^8）の異なる値を表現できる。

### Unicode
- ASCIIよりも多くのbitでアルファベット以外の文字や絵文字も表す。

### RGB
- それぞれの値が8 bit（1 byte, 0-255）なので、3 byteで一色を表現する。
- 1 pixel ＝ 3 byte
